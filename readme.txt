Плагин для WordPress "Калькулятор пени за услуги ЖКХ".

== Description ==

Плагин для WordPress "Калькулятор пени за услуги ЖКХ" не требует никаких настроек и не использует базу данных.

Ставка для расчета пени можно выбирать.

Потребители коммунальных услуг должны оплачивать услугу ежемесячно до 10 числа месяца, следующего за истекшим. Если к этой дате оплата не внесена или внесена частично, то с 31-го дня надо начислить пени.

Размер пени — 1/300 ключевой ставки Центробанка от суммы долга за каждый день просрочки. Начиная с 91-го дня пени рассчитывают в повышенном размере — 1/130 ставки.

Данный калькулятор пени за коммунальные услуги не учитывает день оплаты как день просрочки.


== Installation ==

1. Загрузите плагин "Калькулятор пени за услуги ЖКХ" на сайт WordPress и активируйте его. 

2. Используйте шорткод [un_calculator_peni_teplo] для вставки калькулятора на страницу.
